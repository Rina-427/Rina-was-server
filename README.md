# AWS EC2 Portfolio

## What I built
I launched an EC2 instance and hosted a website using Apache.

Public URL: http://35.73.226.94

## Tech
- AWS EC2
- Amazon Linux 2023
- Apache httpd
- HTML

## Commands
```bash
sudo dnf install httpd -y
sudo systemctl start httpd
sudo systemctl enable httpd
sudo nano /var/www/html/index.html

```
